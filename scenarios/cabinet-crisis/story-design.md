# 三人内阁危机: 剧情走向与时间线设计(主席团专用, 代表不可见)

本文件是DM与中立主席的**导航图, 不是剧本**——代表的行动可以且应当把推演带离任何预设走向;
本文件的价值在于: 判定结果叙述有方向感、时间推进有依据、僵局时知道往哪里丢压力.

## 起始局势(2026-03-15 09:00)
邻国过去一月在边境增兵三个师、炮兵阵地前推40公里. 内阁在"先外交还是先动员"上根本分裂:
总理求稳(议会支持中、军权弱), 国防部长求战备(一线部队忠诚强、议会支持弱), 外交部长握有
副外长级秘密渠道(可靠度强). 危机的核心张力: **对方的军事时间表比我方的决策速度快**.

## 主要剧情走向(五条参考线)

### A. 外交解局线
- 触发倾向: 内阁及时授权外交接触, 且军事动作保持在"对方可解读为防御"的范围内;
- 关键节拍: 秘密渠道激活 → 对方务实派回应 → 第三方地点技术级磋商 → 双方同步降级;
- DM导航: 对方内部有强硬/务实两派. 我方每一次克制都给务实派弹药, 每一次升级都给强硬派借口.
  外交成功不应一帆风顺——至少安排一次"对方拒绝/冷处理"迫使我方内阁再抉择.

### B. 威慑成功线
- 触发倾向: 军事展示有力但守住红线(不越界、不实弹指向对方、统一指挥链);
- 关键节拍: 有节制的演习/部署 → 对方重新计算成本 → 以"共同后撤"名义体面降级;
- DM导航: 威慑有效的前提是**内阁statement一致**; 若内阁公开分裂, 同样的军事动作会被解读为失控而非决心, 转入C线.

### C. 摩擦升级线
- 触发倾向: 边境部队自主权过大、指挥链混乱、或夜间侦察对峙走火;
- 关键节拍: 零星交火 → 伤亡出现 → 国内舆论绑架决策 → 双方被迫增兵;
- DM导航: 升级应当是**一系列小失控的累积**而非一步到位; 每次交火后给内阁一个降级窗口, 看他们接不接.

### D. 内阁内爆线
- 触发倾向: 部长绕过内阁先斩后奏、解职/弹劾/抗命等宪政冲突压过边境议题;
- 关键节拍: 越权指令 → 权威摊牌 → 军队站队/议会介入 → 宪政危机与边境危机叠加;
- DM导航: 内爆期间**对方不会暂停**——指挥链混乱正是对方推进的最佳窗口, 务必让外部压力持续叠加, 不让推演变成纯宫斗.

### E. 战争线(灾难终局)
- 触发倾向: C线连续失控 + 内阁失能, 或任何一方主动越境攻击;
- 关键节拍: 全面动员 → 边境战役 → 推演以"战争爆发"为终局评估收场;
- DM导航: 这是合法结局而非失败——但必须是代表们一连串可追溯的选择导致的, 不允许单次掷骰直接入线.

## 时间线设计(关键节点)

节点是**外部世界的心跳**, 与内阁开会节奏无关地逼近. 机器可读版在 crisis_arcs.yaml 的 timeline 字段,
主席每次危机更新后据此决定跳时(见 04§5); DM判定 takes_effect_at 时应对照节点思考"赶不赶得上".

| 故事时间 | 节点 | 压力含义 |
|---|---|---|
| 03-15 13:00 | 邻国边境实弹演习(预埋arc) | 第一次外部冲击, 检验内阁反应速度 |
| 03-15 15:00 | 议会质询窗口 | 国内政治入场: 内阁必须有可公布的立场 |
| 03-15 18:00 | 邻国晚间新闻/舆论定调 | 对方国内舆论定调后, 务实派回旋空间收窄 |
| 03-15 21:00 | 夜间侦察对峙高峰 | 走火高危时段, 前线自主权问题被放大 |
| 03-16 09:00 | 邻国内阁会议 | 对方路线定调: 此前我方的所有信号在此结算 |
| 03-16 18:00 | 推演终局评估 | 主席按 end_conditions 评估收场 |

## 主席跳时指引
- 会场空转(纯辩论无指令)时, 跳到下一个能制造压力的节点;
- 刚发生重大转折(指令大成功/灾难失败)时, 小步推进(30-60分钟), 给代表消化和反应的空间;
- 不跳过尚未结算的 takes_effect_at(在途行动生效点优先于时间线节点).
